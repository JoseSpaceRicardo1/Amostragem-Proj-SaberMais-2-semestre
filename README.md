# Saber+

`CURSO`
Análise e Desenvolvimento de Sistemas

`DISCIPLINA`
Desenvolvimento de uma Aplicação Interativa

`SEMESTRE`
2º Semestre

O projeto Saber+ consiste no desenvolvimento de uma plataforma web de educação profissionalizante, concebida para operar como um ecossistema colaborativo e gratuito. A aplicação conectará profissionais experientes, dispostos a compartilhar seu conhecimento como instrutores voluntários, a um público de estudantes que busca qualificação de alta qualidade sem custos.
